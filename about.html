Architecture

Subpage-first by default

Each major surface is a standalone HTML page. Shared logic lives in plain JavaScript, and active navigation is derived from window.location.pathname.

Mission

Digital Bharat, direct links, resilient state

The same creator-flow philosophy now powers feed, discover, about, profile, and agent pages. Local profile data, creator drafts, and feed state survive page transitions because they live in browser storage, not router memory.
