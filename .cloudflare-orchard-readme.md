# Orchard Game — Cloudflare Pages Setup

Deploy the Orchard game as a separate Cloudflare Pages project.

## Cloudflare Pages Settings (separate project: orchard-viadecide)

- **Framework preset**: None (custom)
- **Build command**: `cd orchard && npm install && npm run build`
- **Build output directory**: `orchard/dist`
- **Root directory**: `/` (repo root)
- **Node version**: 20

## Environment Variables (set in Cloudflare Pages dashboard)

| Variable | Value |
|----------|-------|
| VITE_FIREBASE_CONFIG | `{"projectId":"...","apiKey":"...","authDomain":"...","appId":"...","firestoreDatabaseId":"...","storageBucket":"...","messagingSenderId":"..."}` |
| VITE_GEMINI_API_KEY | Your Gemini API key |

## Custom Domain
Point to: orchard.viadecide.com

## Update _redirects in main VIA repo
Add this line to VIA's _redirects file:
  /orchard/*  https://orchard.viadecide.com/:splat  200
